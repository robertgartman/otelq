# Code of Conduct

This project adopts the [Contributor Covenant][cc], version 2.1, as its code of
conduct. By participating, you are expected to uphold it. The full text is here:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct.html

To report unacceptable behavior, contact the maintainer at
**robert.gartman.sweden@gmail.com**. All reports are reviewed and handled
confidentially.

[cc]: https://www.contributor-covenant.org
